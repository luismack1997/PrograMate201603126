Problema 61:

Números Cíclicos de Figuras

Enunciado

Números Triangulares, Cuadrados, Pentágonos, Hexágonos, Heptágonos, y Octógonos son todos números de figuras (polígonos) que se generan por las siguientes fórmulas:

  • Triangular: {% latexify 'P_{3,n}=\dfrac{n(n+1)}{2}' math_inline=True %}
  • Cuadrados: {% latexify 'P_{4,n}=n^2' math_inline=True %}
  • Pentagonales: {% latexify 'P_{5,n}=\dfrac{n(3n-1)}{2}' math_inline=True %}
  • Hexagonales: {% latexify 'P_{6,n}=n(2n-1)' math_inline=True %}
  • Heptagonal: {% latexify 'P_{7,n}=\dfrac{n(5n-3)}{2}' math_inline=True %}
  • Octogonal: {% latexify 'P_{8,n}=n(3n-2)' math_inline=True %}

    • El conjunto ordenado de tres números de 4-dígitos: 8128, 2882, 8281, tiene tres propiedades interesantes.
    1. El conjunto es cíclico, de manera que los últimos dos dígitos de cada número son los primeros dos del siguiente (incluyendo el último número con el primero)
    2. Cada tipo de polígono: triangular {% latexify '(P_{3,127}=8128)' math_inline=True %}, cuadrado {% latexify '(P_{4,91}=8221)' math_inline=True %}, y pentagonal {% latexify '(P_{3,127}=8128)' math_inline=True %} es representado por un número diferente en el conjunto.
    3. Este es el único número de 4-dígitos con esta propiedad.

    Encuentre la suma del único conjunto ordenado de seís números de 4-dígitos en donde cada tipo de polígono, es representado por un número diferente en el conjunto

    Solución

    El codigo que resuelve este problema consta de 3 partes. La primera se encarga de verificar si el número ingresado es palíndromo o no. La segunda, pasará el número a binario y finalmente la tercera parte se encargará de verificar si el número en binario también es palíndromo o no. Si es palindromo lo suma y si no lo es pasa al siguiente número.
